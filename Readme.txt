Дневник рациона - приложение, которое поможет следить за потреблением БЖУ. Само приложение будет состоять из двух частей: калькулятор БЖУ и дневник.

В калькуляторе вводишь название продукта и количество грамм, и он автоматически выводит количество БЖУ в продукте на n грамм. После чего можно будет сохранить информацию.

В дневнике будет хранится информация, которая сохраняется из калькулятора.

Приложение будет написано на языке python с использованием библиотеки kivy.

Также будет база данных, в которой хранится количество БЖУ на 100 грамм в определенном продукте.
